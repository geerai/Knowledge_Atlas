## Canonical rule

### Canonical now
For the main product, Family A is canonical.
For contributor operations, Family B is canonical.
For class narrative, Family C is canonical until revised.

### Not yet canonical
Family C pages are not automatically canonical for the main Atlas unless explicitly promoted.

## Implications

1. Main-site navigation should point first to Family A and B pages.
2. Course pages should be linked deliberately, not mixed invisibly into the main IA.
3. Duplicates between Family A and Family C need explicit dispositions.
